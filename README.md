# ThanhGPT Chat Bot
A chatbot using ChatGPT API 
## Installation

Get your API key from OpenAI and paste the key into the ```api/.env``` file

Use the package manager [npm](https://nodejs.org/) to install the required packages.

```bash
npm i
```

## Usage

```bash
node .
```
